# 🍅 Les Tomates du Potager

Collection de variétés anciennes cultivées au jardin familial dans le sud de la France.

## 🌐 Voir le site

👉 **[rjullien.github.io/potager-tomates](https://rjullien.github.io/potager-tomates/)**

## 📍 Localisation

Sud de la France — Zone USDA 9a, climat méditerranéen.

## 📝 Notes

- Toutes les variétés sont des **semences fixées** (pas d'hybrides F1)
- Culture en pleine terre
- Pas de traitement chimique
